# Logo Assets for VERVESTACKS Branding

This directory contains the logo assets used by the VERVESTACKS branding system.

## Required Logo Files:

### 1. vervestacks_logo.png
- **Description**: VERVESTACKS colorful stacked shapes logo
- **Position**: Top left of charts
- **Recommended Size**: 24x24px (small), 48x48px (medium), 96x96px (large)
- **Format**: PNG with transparency

### 2. kanors_logo.png
- **Description**: Kanors EMR logo
- **Position**: Top right of charts
- **Recommended Size**: 24x24px (small), 48x48px (medium), 96x96px (large)
- **Format**: PNG with transparency

## Logo Specifications:
- **File Format**: PNG with alpha channel for transparency
- **Color Space**: RGB or RGBA
- **Resolution**: High resolution (at least 96x96px) for scaling
- **Background**: Transparent
- **Style**: Professional, clean, recognizable at small sizes

## Usage:
The branding system automatically loads these logos and applies them to all charts and visualizations. Logos are positioned automatically based on the branding configuration.

## Note:
If logo files are not found, the branding system will gracefully handle the missing assets and continue without logos.
